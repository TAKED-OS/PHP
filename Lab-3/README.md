### Контрольные вопросы

1.В чем разница между циклами for, while и do-while? В каких случаях лучше использовать каждый из них?

#### For

Используется, когда заранее известно количество повторений.

Пример:

```
for ($i = 0; $i < 5; $i++) {
    echo $i;
}
```

Структура: for (начало; условие; изменение)

- Применяется когда:
  - есть счетчик
  - известно сколько раз выполнять цикл

#### While

Используется, когда количество повторений заранее неизвестно.

Пример:

```
$i = 0;

while ($i < 5) {
    echo $i;
    $i++;
}
```

- Применяется когда:
  - цикл зависит от какого-то условия
  - нет точного количества повторений

#### Do-While

Главная особенность — цикл выполняется минимум один раз.

Пример:

```
$i = 0;

do {
    echo $i;
    $i++;
} while ($i < 5);
```

- Применяется когда:
  - код должен выполниться хотя бы один раз
 
2.Как работает тернарный оператор ? : в PHP?

Тернарный оператор — это короткая форма if-else.

Общий вид:
` условие ? значение1 : значение2; `

Если условие истинно -> возвращается значение1

Если ложно -> значение2.
